# OKF Agriculture Knowledge Graph

This directory contains the schemas, ontology files, and entity relations defining the agricultural knowledge graph.

## Folder Structure

* `schema/`: Ontology schema definitions (e.g., Turtle format `.ttl` or JSON-LD).
* `data/entities/`: Node entity configuration files (JSON or CSV).
* `data/relations/`: Edge relation configuration files (JSON or CSV).
* `data/raw/`: Raw files or scrapings to ingest.
* `scripts/`: Python ingestion and query scripts.
